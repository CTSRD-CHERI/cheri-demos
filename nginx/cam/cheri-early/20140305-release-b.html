Second BERI/CHERI documentation release for early adopters (20140305)

Description

This is our second "early adopter" documentation release, which includes the BERI Hardware Reference, BERI Software Reference, CHERI Architecture Guide, and CHERI User's Guide. The introduction of each document includes a brief version history describing changes from earlier versions. The CHERI Documentation Roadmap is the recommended starting point.
