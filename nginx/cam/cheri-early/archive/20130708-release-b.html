CHERI Test Suite 1.0 (20130708)

Description

This is the first release of the CHERI Test Suite, v1.0. The test suite exercises ISA-level functionality in the CHERI CPU, and is intended for use with the CHERI and CHERI2 simulators. It can also be run against the gxemul MIPS emulator.

This release has been tested from an Ubuntu 12.04.1 on x86_64.
