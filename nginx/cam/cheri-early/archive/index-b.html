CHERI archival releases

This is an archival version of the original CHERI early adopters' website which provides access to 2013 early releases to MIT/LL, MITRE, and other CRASH/MRC-programme users. The current website can be found here.

This website provides information, hardware designs, and software for early-access users of the CHERI microprocessor. This includes architecture, platform reference, and user guides, FPGA bitfiles to be flashed onto development boards, and software images necessary to exercise and experiment with the CHERI architecture.

SRI International and the University of Cambridge are able to provide only limited support for externel CHERI deployment. However, we encourage you to contact us if any issues are encountered, and we will work to support a growing user community as resources permit.

WARNING

CHERI and its software are highly experimental; early adopters are warned that correct operation is unlikely (despite the best of intentions by its developers). Further, CHERI units may interfere with other devices on the network, and have done so in the past, due to use of experimental network features, device driver bugs, and so on.

Users may wish to employ caution when deploying CHERI devices in production network environments, and will ideally place them behind existing firewalls. It is also possible (and in fact, fairly easy) to cause permanent damage to FPGA development boards -- e.g., by damaging the on-board flash (etc), which is an expensive scenario best avoided.

Current releases

22 July 2013 - Annabella release of CHERI DE4 bitfile, cherictl, FreeBSD/BERI, CheriBSD
This is our second "early adopter" release of a CHERI bitfile for the Terasic DE-4, along with corresponding cherictl tool, FreeBSD/BERI, and CheriBSD software images.
8 July 2013 - CHERI Test Suite 1.0
This is our first release of the CHERI Test Suite, which is used to validate ISA-level functionality in the CHERI processor.
7 April 2013 - Ambrunes release of CHERI DE4 bitfile, cherictl, FreeBSD/BERI, CheriBSD
This is our first "early adopter" release of a CHERI bitfile for the Terasic DE-4, along with corresponding cherictl tool, FreeBSD/BERI, and CheriBSD software images.
20 March 2013 - Adobe Illustrator layout for DE4/MTL tablet plexiglass (non-battery)
This Adobe Illustrator file is used by Cambridge when laser-cutting face plates and related parts for the battery-free DE4/MTL tablet.
16 March 2013 - First documentation release for CHERI early adopters
This is our first "early adopter" documentation release, which includes the CHERI Architecture Guide, CHERI Platform Reference, and CHERI User's Guide. This is a precursor to a first binary bitfile and software release.
