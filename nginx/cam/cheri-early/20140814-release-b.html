Updated CHERI bitfile (20140814)

Description

This is an updated CHERI bitfile that includes many enhancements since the Ambrunes release, including countless bug fixes and performance improvements, USB support, and ISA enhancements such as experimental support for capability cursors (offsets). With the exception of cursor support, this bitfile corresponds to the ISA described in the CHERI Instruction-Set Architecture (UCAM-CL-TR-850). report.
