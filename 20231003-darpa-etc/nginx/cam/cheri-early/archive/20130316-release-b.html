First CHERI documentation release for early adopters (20120316)

Description

This is our first "early adopter" documentation release, which includes the CHERI Architecture Guide, CHERI Platform Reference, and CHERI User's Guide. This is a precursor to a first binary bitfile and software release. The CHERI Documentation Roadmap is the recommended starting point.
