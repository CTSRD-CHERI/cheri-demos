CHERI, CheriBSD Ambrunes Release (20130407)

Description

This is the first early adopter's release of CHERI and CheriBSD, "Ambrunes". The release includes a bitfile suitable for use on the Terasic DE4, the cherictl mananagement tool, and software images for FreeBSD/BERI and CheriBSD suitable for use on the hardware prototype.

This release has been tested from an Ubuntu 12.04.1 on x86_64 with Quartus 12.1 installed. Other versions of Linux x86_64 are likely to work assuming that Quartus 12.1 is installed.

The FreeBSD/BERI and CheriBSD snapshots in this release correspond to the FreeBSD Project perforce branches //depot/projects/ctsrd/beribsd/... and //depot/projects/ctsrd/cheribsd/... at change 226199.
