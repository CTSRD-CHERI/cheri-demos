BERI Test Suite 1.1 (20140301)

Description

This is the second release of the BERI Test Suite, v1.1. The test suite exercises MIPS and CHERI ISA functionality at the instruction level in the BERI CPU, and is intended for use with the BERI and BERI2 simulators. It can also be run against the gxemul MIPS emulator to validate MIPS ISA behaviour.

This release has been tested from an Ubuntu 12.04.1 on x86_64.
