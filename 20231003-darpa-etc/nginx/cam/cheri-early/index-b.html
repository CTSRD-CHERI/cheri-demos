BERI/CHERI early adopters website

This website provides information, hardware designs, and software for early-access users of the BERI microprocessor and its CHERI ISA extensions. This includes architecture, platform reference, and user guides, FPGA bitfiles to be flashed onto development boards, and software images necessary to exercise and experiment with BERI and CHERI.

SRI International and the University of Cambridge are able to provide only limited support for externel BERI/CHERI deployment. However, we encourage you to contact us if any issues are encountered, and we will work to support a growing user community as resources permit.

WARNING

BERI/CHERI and its software are highly experimental; early adopters are warned that correct operation is unlikely (despite the best of intentions by its developers). Further, BERI units may interfere with other devices on the network, and have done so in the past, due to use of experimental network features, device driver bugs, and so on.

Users may wish to employ caution when deploying BERI devices in production network environments, and will ideally place them behind existing firewalls. It is also possible (and in fact, fairly easy) to cause permanent damage to FPGA development boards -- e.g., by damaging the on-board flash (etc), which is an expensive scenario best avoided.

Current releases

If you have previously downloaded our Annabella and Ambrunes 2013 early releases as part of the DARPA CRASH programme, you can find them here.

13 August 2014 - CHERI bitfile update
This bitfile update is replaces the Ambrunes bitfile previously made available to CHERI early adopters in the DARPA CRASH and MRC research programmes.
13 March 2014 - BERI/CHERI source code
This is the first BERI/CHERI source-code release, including both Bluespec source code for the BERI1/BERI2 processors, supporting Bluespec libraries, and C-language simulation libraries and tools. It corresponds roughly to our 1 March 2014 BERI test suite release, and 5 March 2014 documentation release.
5 March 2014 - Documentation updates
Updates to the BERI Hardware Reference, BERI Software Reference, CHERI Architecture Document, CHERI User's Guide, and CHERI Documentation Roadmap to reflect changes since the March 2013 drops of the same documents.
1 March 2014 - BERI Test Suite 1.1
This is our second release of the CHERI Test Suite, which is used to validate MIPS and CHERI ISA instruction-level behaviour in BERI.
